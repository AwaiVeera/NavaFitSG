### Prerequisites
- Node.js 18+ and npm 9+
- Modern web browser
- Firebase account (for backend features)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/navafit/navafit-sg.git
cd navafit-sg
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm start
```

4. **Open your browser**
The development server will automatically open http://localhost:3000
